| 标题栏                导航区(navBar)|  插槽： title、navBar

| |   表单(form)                 |   |    插槽：formHeading(form之前)、fromTrailing(form尾部，submit 之后)、beforeSubmit

+------------------------------------+
| 工具栏(toolbar)                     |  插槽：toolbar、error

|                                    |

|                                    |
+------------------------------------+
| 底部工具栏(bottomToolbar)            | 插槽：bottomToolbar
+------------------------------------+
|                            分页     |
